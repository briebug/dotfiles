# BrieBug dotfiles

A set of configuration files.

## Getting started

Setup is not automated (yet). For now:

```
cd ~
git clone git@github.com:briebug/dotfiles.git
ln -s ~/dotfiles/zsh .zsh
ln -s ~/dotfiles/zshrc .zshrc
```

Link any other rc files that you wish to use.
